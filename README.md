# mask_stats
<b>Python3 script to compare ALMA and SoFiA masks and evaluate the fraction of masked pixels in an ALMA cube</b>

The script is run as "<i>local_path/mask_stats.py <ALMA_data_set> <ALMA mask> <SoFiA mask></i>"

The file format used is FITS.

The first line in the script must then be changed to point to your local python3 installation and the script must be made executable with "<i>chmod +x mask_stats.py</i>"

Alternatively one can run the script as "<i>python3 local_path/mask_stats.py <ALMA_data_set> <ALMA mask> <SoFiA mask></i>"
